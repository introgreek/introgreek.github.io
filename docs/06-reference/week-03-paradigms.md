## Verbs

### Aorist Passive

| Person | Singular | Plural |
| ----- | ----- | ----- |
| 1st | `-ην` | `-ημεν` |
| 2nd | `-ης` | `-ητε`|
| 3rd | **-η** | **-ησαν** |

### Summary of Aorist Forms

| Voice | 3rd Singular | 3rd Plural |
| ----- | ----- | ----- |
| Active | ἐκέλευσε(ν) | ἐκέλευσαν |
| Middle | ἐκελεύσατο | ἐκελεύσαντο |  
| Passive | ἐκελεύσθη | ἐκελεύσθησαν |
